# 快捷键大全

## typroa

```bash
command + 0 # 清除格式，变成正文
command + / # 切换成源码模式
```

## Mac OS

```bash
ctrl + command + Q # 快捷锁屏
command + r # 刷新
command + shift + 4 # 截屏 
command + shift + 3 # 截全屏 
```

## vscode

```bash
command + / # 拆分编辑器
shift + tab # 反向 tab
```
