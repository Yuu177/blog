## 名词介绍

### PATH

PATH 说简单点就是一个字符串变量，当输入命令的时候 LINUX 会去查找 PATH 里面记录的路径。所以，path 配置的路径下的文件可以在任何位置执行，并且可以通过 which 可执行文件 命令来找到该文件的位置。

- 查看 PATH

`echo $PATH`

| lsof -i:xx | 查看哪个程序占用了 xx 端口号                                 |
